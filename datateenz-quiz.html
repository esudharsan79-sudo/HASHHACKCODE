📘 What is JavaScript?

JavaScript is a programming language used to make web pages interactive. It allows users to click buttons, submit forms, and update content dynamically without reloading the page.

Example:

document.getElementById("demo").innerText = "Hello User";

📘 What is a Variable?

A variable is used to store data. It acts like a container that holds values which can be reused and modified later in the program.

Example:

let name = "John";
