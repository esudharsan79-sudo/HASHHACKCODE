# Datateenz - JavaScript Learning Module

## 📌 Project Overview

This project is a simple interactive web application built using HTML, CSS, and JavaScript.
It is designed to help beginners understand basic JavaScript concepts through explanations and a quiz.

---

## 🎯 Objectives

* Understand JavaScript fundamentals
* Learn how JavaScript handles user interactions
* Understand variables and data storage
* Practice concepts using a quiz

---

## 📘 Topics Covered

* What is JavaScript
* What is a Variable
* Basic DOM interaction

---

## ❓ Features

* Interactive quiz with multiple questions
* Timer (60 seconds)
* Shows correct and wrong answers
* Displays explanation after each answer
* Score tracking (Correct / Wrong)
* Progress bar
* Reset option

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## ▶️ How to Run

1. Download the project file
2. Open the `.html` file in any browser
3. Start the quiz

---

## 📊 Functionality

* Each question can be answered only once
* After selecting an option:

  * Correct answer is highlighted
  * Wrong answer is marked
* Timer ends the quiz automatically
* Final score is displayed at the end
